# JHToolsModule
工具类集合，包括类别，宏，工具视图，base视图，自用

依赖  Masonry   LNRefresh  EasyShowView  SDWebImage

##  安装
### 1.手动添加:<br>
*   1.将 JHToolsModule 文件夹添加到工程目录中<br>
*   2.导入 JHUITools.h

### 2.CocoaPods:<br>
*   1.在 Podfile 中添加 pod 'JHToolsModule'<br>
*   2.执行 pod install 或 pod update<br>
*   3.导入 JHUITools.h
